# Garden Supplies Store - Интернет-магазин садовых товаров

Полнофункциональное веб-приложение интернет-магазина садовых принадлежностей, реализованное на React с использованием Redux Toolkit, React Router и React Hook Form.

## Структура проекта

Проект состоит из двух основных частей:

- `client/` - React-приложение (фронтенд)
- Бэкенд должен быть установлен отдельно (см. https://github.com/Ray3en/telran-backend)

## Установка и запуск

### 1. Установка бэкенда

Следуйте инструкциям в репозитории бэкенда:
https://github.com/Ray3en/telran-backend

**Важно!** Исходный код бэкенда менять нельзя.

Убедитесь, что бэкенд запущен на порту 3333.

### 2. Установка фронтенда

Перейдите в папку `client` и установите зависимости:

```bash
cd client
npm install
```

### 3. Запуск фронтенда

```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000

## Технологии

- **React 18.3.1** - UI библиотека
- **Redux Toolkit 2.2.7** - управление состоянием
- **React Router DOM 6.28.0** - маршрутизация
- **React Hook Form 7.53.0** - управление формами
- **React Redux 9.1.3** - интеграция Redux с React
- **CSS Modules** - модульная стилизация

## Архитектура

Проект следует модульной архитектуре:

- `components/` - переиспользуемые компоненты
  - Каждый компонент находится в своей папке
  - Содержит `.jsx` файл и модульные стили `.module.css`
  - Имена компонентов используют постфиксы (Page, Card, Button, Slider и т.д.)
- `pages/` - страницы приложения
  - Каждая страница в своей папке с модульными стилями
- `store/` - Redux хранилище
  - `slices/` - Redux Toolkit slices для управления состоянием
  - `index.js` - конфигурация store
- `assets/` - медиа файлы (иконки, изображения

- **PrimaryButton** - универсальная кнопка с вариантами (addToBasket, hero)

## Основные функции

- Просмотр категорий и товаров
- Фильтрация товаров по цене и наличию скидки
- Сортировка товаров (по умолчанию, новинки, цена)
- Просмотр товаров со скидкой
- Добавление товаров в корзину с выбором количества
- Управление корзиной (изменение количества, удаление)
- Оформление заказа с формой контактов
- Форма заявки на скидочный купон (5% на первый заказ)
- Адаптивный дизайн для мобильных устройств

## Redux Store

### Slices:

- **goodsSlice** - управление товарами
  - `loadAllGoods` - загрузка всех товаров
  - `loadGoodById` - загрузка товара по ID
  - `clearSelectedGood` - сброс текущего товара
- **sectionsSlice** - управление категориями
  - `loadAllSections` - загрузка всех категорий
  - `loadSectionGoods` - загрузка товаров категории
  - `clearSectionData` - сброс данных категории
- **basketSlice** - управление корзиной
  - `addToBasket` - добавление товара в корзину
  - `removeFromBasket` - удаление товара из корзины
  - `changeQuantity` - обновление количества товара
  - `emptyBasket` - очистка корзины
  - `getBasketItemsCount` - селектор количества товаров

## API Endpoints

Все запросы проксируются на бэкенд (localhost:3333):

- `GET /categories/all` - получить все категории
- `GET /categories/:id` - получить товары категории
- `GET /products/all` - получить все товары
- `GET /products/:id` - получить товар по ID
- `POST /sale/send` - отправить заявку на купон
- `POST /order/send` - отправить заказ
